# sales-performance-analysis
# Tableau de bord d'analyse des performances de vente

## Description du projet
Ce projet consiste à analyser des données de ventes afin de transformer les informations brutes en **insights exploitables** pour soutenir la prise de décision.

L’objectif principal est de construire **un tableau de bord interactif** pour suivre les performances des ventes, identifier les tendances et détecter les opportunités d’amélioration.

---

## Objectifs
- Suivi des KPI de ventes
- Analyse géographique et par catégorie de produits
- Détection des tendances et anomalies (annulations, saisonnalité)
- Aide à la prise de décision basée sur les données

---

## Principaux résultats
- 💰 **Chiffre d’affaires total :** 2 millions de dollars  
- 📦 **Nombre total de commandes :** 500  
- 🛒 **Unités vendues :** 3 109  
- 📊 **Valeur moyenne d'une commande :** ~ 4 800 $

- 🌍 **Région la plus performante :** Sud  
- 🖥 **Répartition par catégorie :** Électronique 62 %, Accessoires 38 %  
- ⚠️ **Taux d’annulation :** 49 % (≈ 245 commandes), plus élevé vers la fin de l’année
